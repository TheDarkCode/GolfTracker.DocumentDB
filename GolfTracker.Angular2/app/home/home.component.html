This is an Angular 2 real-world application ported from the original Angular 1 application that can use Azure DocumentDB or MongoDB out of the box.

The Golf Tracker application allows golfers to track their scores as they play golf.

Directions

  1. Golf Clubs - first enter the golf clubs, courses and tees.
  2. Manage Golfers - then enter a golfer
  3. Add Round - and finally add a round of golf for the golfer

The Architecture

  • UI - light-weight Angular 2 framework
  • Data - the application is flexible to handle different data stores. The Angular services point to RESTful endpoints for data. Currently it is setup to use either Azure DocumentDB or MongoDB.

Updates

  • 8/5/2016 - extracted the embedded HTML into their own components to be used as directives in the main HTML.
  • 7/28/2016 - This application has been updated to 2.0.0-rc.4.

System Requirements

  • NodeJs and NPM
  • Either MongoDB or Azure DocumentDB
